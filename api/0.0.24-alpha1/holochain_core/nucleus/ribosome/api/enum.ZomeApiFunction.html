Enumeration of all the Zome Functions known and usable in Zomes. Enumeration can convert to str.

Variants

MissingNo

Error index for unimplemented functions

Abort

Abort is a way to receive useful debug info from assemblyscript memory allocators message: mem address in the wasm memory for an error message filename: mem address in the wasm memory for a filename line: line number column: column number

Debug

send debug information to the log debug(s: String)

CommitAppEntry

Commit an app entry to source chain commit_entry(entry_type: String, entry_value: String) -> Address

GetAppEntry

Get an app entry from source chain by key (header hash) get_entry(address: Address) -> Entry

UpdateEntryRemoveEntryInitGlobals

Init Zome API Globals hc_init_globals() -> InitGlobalsOutput

Call

Call a zome function in a different zome or dna via a bridge hc_call(zome_name: String, cap_token: Address, fn_name: String, args: String);

LinkEntries

Create a link entry

GetLinks

Retrieve links from the DHT

Query

Query the local chain for entries

EntryAddress

Pass an entry to retrieve its address the address algorithm is specific to the entry, typically sha256 but can differ entry_address(entry: Entry) -> Address

Send

Send a message directly to another node

Sleep

Allow a specified amount of time to pass

RemoveLink

Commit link deletion entry

CryptoSignOneTime

Sign a block of data with a one-time key that is then shredded

VerifySignature

Verify that a block of data was signed by a given public key

KeystoreList

Retrieve a list of identifiers of the secrets in the keystore

KeystoreNewRandom

Create a new random seed Secret in the keystore

KeystoreDeriveSeed

Derive a new seed from an existing seed in the keystore

KeystoreDeriveKey

Create a new key (signing or encrypting) as derived from an existing seed in the keystore

KeystoreSign

Sign a block of data using a key in the keystore

KeystoreGetPublicKey

Get the public key for a given secret

CommitCapabilityGrant

Commit a capability grant to the source chain

CommitCapabilityClaim

Commit a capability grant to the source chain

EmitSignal

Send a DNA defined signal to UIs and other listeners
