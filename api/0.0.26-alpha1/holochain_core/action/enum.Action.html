All Actions for the Holochain Instance Store, according to Redux pattern.

Variants

Commit((Entry, Option<Address>, Vec<Provenance>))

Writes an entry to the source chain. Does not validate, assumes entry is valid.

Hold(EntryWithHeader)

Adds an entry to the local DHT shard. Does not validate, assumes entry is valid.

AddLink(LinkData)

Adds a link to the local DHT shard's meta/EAV storage Does not validate, assumes link is valid.

CrudStatus((EntryWithHeader, CrudStatus))RemoveLink(Entry)InitNetwork(NetworkSettings)

Create a network proxy instance from the given NetworkSettings

ShutdownNetwork

Shut down network by sending JsonProtocoll::UntrackDna, stopping network thread and dropping P2pNetwork instance

Publish(Address)

Makes the network PUT the given entry to the DHT. Distinguishes between different entry types and does the right thing respectively. (only publish for AppEntryType, publish and publish_meta for links etc)

GetEntry(GetEntryKey)

Get an Entry on the network by address

RespondGet((QueryEntryData, Option<EntryWithMetaAndHeader>))

Lets the network module respond to a Get request. Triggered from the corresponding workflow after retrieving the requested entry from our local DHT shard.

RespondFetch((FetchEntryData, Vec<EntryAspect>))

Lets the network module respond to a FETCH request. Triggered from the corresponding workflow after retrieving the requested entry from our local DHT shard.

HandleGetResult((Option<EntryWithMetaAndHeader>, GetEntryKey))

We got a response for our get request which needs to be added to the state. Triggered from the network handler.

UpdateEntry((Address, Address))
RemoveEntry((Address, Address))
GetEntryTimeout(GetEntryKey)
GetLinks((GetLinksKey, Option<CrudStatus>, GetLinksNetworkQuery))

get links from entry address and link_type name Last string is the stringified process unique id of this hdk::get_links call.

GetLinksTimeout(GetLinksKey)RespondGetLinks((QueryEntryData, GetLinksNetworkResult, String, String))HandleGetLinksResult((GetLinksNetworkResult, GetLinksKey))SendDirectMessage(DirectMessageData)

Makes the network module send a direct (node-to-node) message to the address given in DirectMessageData

SendDirectMessageTimeout(String)

Makes the direct message connection with the given ID timeout by adding an Err(HolochainError::Timeout) to NetworkState::custom_direct_message_replys.

ResolveDirectConnection(String)

Makes the network module forget about the direct message connection with the given ID. Triggered when we got an answer to our initial DM.

GetValidationPackage(ChainHeader)

Makes the network module DM the source of the given entry and prepare for receiveing an answer

HandleGetValidationPackage((Address, Option<ValidationPackage>))

Updates the state to hold the response that we got for our previous request for a validation package. Triggered from the network handler when we get the response.

HandleCustomSendResponse((String, Result<String, String>))

Updates the state to hold the response that we got for our previous custom direct message. Triggered from the network handler when we get the response.

RespondAuthoringList(EntryListData)

Sends the given data as JsonProtocol::HandleGetAuthoringEntryListResult

RespondGossipList(EntryListData)

Sends the given data as JsonProtocol::HandleGetGossipEntryListResult

InitializeChain(Dna)

initialize a chain from Dna not the same as init may call init internally

ReturnInitializationResult(Result<Initialization, String>)

return the result of an InitializeChain action the result is an initialization structure which include the generated public token if any

SignalZomeFunctionCall(ZomeFnCall)

Gets dispatched when a zome function call starts. There is no reducer for this action so this does not change state (hence "Signal"). Is received as signal in the nodejs waiter to attach wait conditions.

ReturnZomeFunctionResult(ExecuteZomeFnResponse)

return the result of a zome WASM function call

ReturnValidationResult(((ProcessUniqueId, Address), ValidationResult))

A validation result is returned from a local callback execution Key is an unique id of the calling context and the hash of the entry that was validated

ReturnValidationPackage((ProcessUniqueId, Result<ValidationPackage, HolochainError>))

A validation package was created locally and is reported back to be added to the state

AddPendingValidation(PendingValidation)

An entry could not be validated yet because dependencies are still missing. This adds the entry to nucleus state's pending list.

RemovePendingValidation((Address, ValidatingWorkflow))

Clear an entry from the pending validation list

Ping

No-op, used to check if an action channel is still open
